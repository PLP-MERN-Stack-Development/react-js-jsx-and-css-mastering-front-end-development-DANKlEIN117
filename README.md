#  React + Tailwind CSS Project

##  Overview

This project is a modern **React.js** application built using **Vite** and styled with **Tailwind CSS**. It showcases clean component architecture, responsive design, and smooth user experience ‚Äî all powered by lightweight and blazing-fast tools.

This app serves as a template or portfolio project for anyone diving into **modern front-end development** using **React** and **Tailwind**.

---

##  Features

- ** Fast Setup with Vite** ‚Äì instant dev server, hot reload
- ** Tailwind CSS Styling** ‚Äì utility-first, fully responsive design
- ** Reusable Components** ‚Äì Navbar, Buttons, Cards, and Footer
- ** Dark/Light Mode Toggle** ‚Äì seamless theme switching
- ** Local Storage Support** ‚Äì keeps data saved even after refresh
- ** API Integration** ‚Äì connects with external APIs
- ** Deployed on Netlify** ‚Äì free hosting with auto builds

##  Deployment on Netlify

### Option 1: Deploy via GitHub (Recommended)

1. Push your project to GitHub
2. Go to [Netlify](https://app.netlify.com/)
3. Click **"Add New Site" ‚Üí "Import an existing project"**
4. Connect your GitHub account and select your repository
5. In **Build settings**, configure:
   - **Build command:** `npm run build`
   - **Publish directory:** `dist`
6. Click **Deploy site**
